# BNB Alpha Agent

BNB Alpha Agent is a lightweight autonomous trading agent prototype built on BNB Chain.

The project provides market signals, risk assessment, portfolio monitoring, and autonomous agent status APIs through a FastAPI backend.

## Core Features

* Multi-token trading signals
* Risk scoring
* Portfolio monitoring
* Agent status monitoring
* Swagger API documentation

## Target Track

Autonomous Trading Agents

## Technology Stack

* Python
* FastAPI
* Swagger UI
* GitHub
